## Cốt lõi (Core)
direction: Hướng trượt.

    string: 'horizontal' (mặc định), 'vertical'.

speed: Tốc độ chuyển slide (ms).

    number: 300 (mặc định).

spaceBetween: Khoảng cách giữa các slide (px).

    number: 0.

slidesPerView: Số slide hiển thị.

    number: 1 (mặc định), hoặc 3, 4...

    string: 'auto'.

centeredSlides: Căn slide active vào giữa.

    boolean: false (mặc định).

loop: Lặp lại vô tận.

    boolean: false (mặc định).

breakpoints: Tùy chỉnh responsive theo kích thước màn hình.

    object: { 640: { slidesPerView: 2 }, 1024: { slidesPerView: 3 } }.

## Modules
autoplay: Tự động chạy.

    boolean: true.

    object: { delay: 2500, disableOnInteraction: false }.

navigation: Nút điều hướng (next/prev).

    boolean: true.

    object: { nextEl: '.next-btn', prevEl: '.prev-btn' }.

pagination: Dấu chấm/số phân trang.

    boolean: true.

    object: { clickable: true, type: 'bullets' }. ('bullets', 'fraction', 'progressbar').

effect: Hiệu ứng chuyển slide.

    string: 'slide' (mặc định), 'fade', 'cube', 'coverflow', 'flip', 'cards'.

## Tương tác (Interaction)
grabCursor: Đổi con trỏ thành hình bàn tay.

    boolean: false (mặc định).

mousewheel: Điều khiển bằng con lăn chuột.

    boolean: false (mặc định).

keyboard: Điều khiển bằng bàn phím.

    boolean: false (mặc định).

## Sự kiện (Events)
    on: Gắn các hàm xử lý vào sự kiện của Swiper.

    object: { init: () => { ... }, slideChange: () => { ... } }.

Các sự kiện phổ biến: init, slideChange, click, reachEnd.
